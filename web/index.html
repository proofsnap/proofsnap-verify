What Gets Verified

Five independent cryptographic and forensic checks on every evidence package.

Digital Signature

RSA-4096 signature verification ensures the manifest was created by ProofSnap and has not been altered.

SHA-256 Hashes

Every file in the package is re-hashed and compared against the manifest to detect any tampering.

Hash Chain

Forensic log hash chain (ISO/IEC 27037) confirms the sequence of capture operations was not modified.

Bitcoin Timestamp

OpenTimestamps proof anchored to Bitcoin blockchain. Use the CLI tool for full blockchain verification.

eIDAS Timestamp

EU qualified timestamp (RFC 3161) structural validation. Valid under eIDAS Regulation 910/2014.
