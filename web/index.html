What Gets Verified

Independent cryptographic checks across three input modes. No vendor lock-in — verify ProofSnap evidence and qualified timestamps from any EU TSA.

Digital Signature

RSA-4096 signature on the ProofSnap manifest, plus RSA / ECDSA signatures inside any RFC 3161 timestamp token.

SHA-256 Hash Match

Every file in the package — or your standalone file — is re-hashed and compared with the timestamp's message imprint.

Hash Chain

Forensic log hash chain (ISO/IEC 27037) confirms the sequence of capture operations was not modified after the fact.

Bitcoin Timestamp

OpenTimestamps proof anchored to Bitcoin blockchain (full chain verification via the CLI tool).

eIDAS Qualified Timestamp

RFC 3161 token signature, certificate chain, TSA identity, policy OID and (with LTV) OCSP revocation status.
